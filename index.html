โดยใช้ Google Apps Script เป็นตัวกลาง

📌 วิธีตั้งค่า: สร้าง Google Apps Script แล้วนำ Deployment URL มากรอกด้านล่าง
✅ คัดลอกโค้ดแล้ว! ไปวางใน Google Apps Script แล้ว Deploy
ขั้นตอน:
1. ไปที่ script.google.com → สร้างโปรเจกต์ใหม่
2. ลบโค้ดเดิมทิ้ง แล้ววางโค้ดที่คัดลอก
3. กด Deploy → New Deployment → Web App
4. ตั้ง "Execute as: Me" และ "Who has access: Anyone"
5. คัดลอก URL มากรอกด้านบน แล้วกด บันทึก
✅ บันทึกสำเร็จ
